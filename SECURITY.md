# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 4.1.x   | :white_check_mark: |
| < 4.0.0 | :x:                |

## Reporting a Vulnerability

If you discover a security vulnerability, please report it via GitHub Issues (for non-sensitive bugs) or contact the maintainer directly if it involves private data handling.

As this is a terminal-based tool focusing on privacy, we take security reports seriously.
